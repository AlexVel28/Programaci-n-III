-----------FUNCIÓN DEL PROGRAMA-------------
Este programa en C++ permite ingresar números de forma dinámica y clasificarlos automáticamente en diferentes estructuras de datos según su tipo:
	- Pila (LIFO): almacena números impares
	- Cola (FIFO): almacena números pares
	- Lista enlazada: almacena números negativos

El objetivo del programa es desarrollar un programa que implemente estructuras dinámicas (pila, cola y lista enlazada), aplicando lógica de decisión, validación de datos y manejo correcto de memoria.

----------FUNCIONALIDADES-----------

Ingreso de datos:
	- Permite ingresar números de forma dinámica
	- Finaliza con el valor especial -1
Clasificación automática:
	- Número par → Cola
	- Número impar → Pila
	- Número negativo (excepto -1) → Lista enlazada
