Este ejercicio consiste en un sistema integral desarrollado en C++ que implementa y gestiona múltiples estructuras de datos lineales y no lineales de forma manual. 
El programa permite la entrada dinámica de números enteros, su clasificación automática basada en reglas lógicas y el análisis estadístico de la información procesada.

Estructuras usadas:
	- PILA
	- COLA
	- LISTA SIMPLEMENTE ENLAZADA
	- LISTA DOBLEMENTE ENLAZADA
	- ARBOL BINARIO DE BUSQUEDA

Explicación Lógica:
El sistema opera bajo un flujo de clasificación automática y procesamiento diferido:
	- Clasificación en tiempo real
	- Registro de historial
	- Construcción del BST
	- Procesamiento de negativos
	- Reorganización de memoria

Instrucciones de ejecución:
	1. Abrir el programa
	2. Compilar y ejecutar el programa
	3. Ejecutar el binario del programa
	4. Realizar las tareas deseadas.
